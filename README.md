# Binance Futures Testnet Trading Bot

A production-ready Python trading bot for Binance Futures Testnet with comprehensive error handling, logging, and validation.

## Features

- ✅ **MARKET Orders** - Instant execution at current market price
- ✅ **LIMIT Orders** - Set specific price for execution  
- ✅ **BUY/SELL Support** - Both sides supported
- ✅ **CLI Interface** - Command-line and interactive modes
- ✅ **Comprehensive Logging** - All API requests/responses logged
- ✅ **Input Validation** - Prevents invalid orders
- ✅ **Error Handling** - Handles API errors, network failures
